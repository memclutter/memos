# memos (project)

This is the project entry for **the OS itself**. The `memos` repository is the
personal operating system described at the repo root — its rules, agent skills,
and helper scripts.

Unlike every other project, `memos` has **no `repo/` submodule**: a repository
cannot contain itself as a submodule. Its source *is* the OS repo root. This
folder exists so the OS can be managed like any other project — with its own
tasks and docs.

- **Source:** the repository root (this same repo,
  `git@github.com:memclutter/memos.git`).
- **Tasks:** [tasks/](tasks/) — work on the OS itself (rules, skills, scripts).
- **Docs:** [docs/](docs/).

See the root [README.md](../../README.md) and [AGENTS.md](../../AGENTS.md) for
what the OS is and how it is structured.
