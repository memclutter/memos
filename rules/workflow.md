# Agent workflow

The working agreement for any AI agent operating in this repo.

- Read `AGENTS.md` first; it indexes every rule in `rules/`. Open the relevant
  rule file before acting on its topic.
- Keep records current as part of doing the work: each task iteration adds a
  `log/` folder with its artifacts and summary; a new recurring tool choice
  updates the rules.
- Prefer the base stack ([stack.md](stack.md)); when you deviate, write down why.
- Use Conventional Commits for every commit ([git.md](git.md)).
- Ask the owner (in Russian) when a decision is genuinely theirs; otherwise pick
  the documented default and proceed.
- When `rules/` changes, update the matching paragraph in `AGENTS.md`.
